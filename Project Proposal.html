Project Description

We will be exploring the virtual world of Minecraft Pi. We will introduce a first-time setup on how to install Minecraft Pi formerly and learning how to control the player, manually build with blocks and use the Python interface to make up the world (Using Python 3).We will also make our Minecraft Pi and tutorials available on the Internet.
Why this is an IoT project

This project can represent IoT because we are applying knowledge throughout game physics by using programming and code to create a working functionality in game with a Raspberry Pi that will be connected through the Internet. Plan/expect to learn

We will be able to recognize programming methods through Python 3 and learn along the way how it affects our methods and decision making. In summary we plan/expect to learn

  • Python 3 programming
  • Python 3 libraries
  • Raspberry Pi
  • Minecraft Pi
  • IoT setup (i.e. making the Raspberry Pi containing Minecraft Pi available online)
  • How to work as a team

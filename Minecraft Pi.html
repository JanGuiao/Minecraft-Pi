Getting Started

Install PythonIDLE, the latest version:

IDLEPython

Download "Minecraft v1.12 Windows Starter Kit" zip file:

Windows Starter Kit

Starting with the "AdventuresInMinecraft-PC", make sure you extract the folder to your desktop.

Next, open PythonIDLE, create a new file (file, top-right) and save the file to your "Windows Starter Kit" folder, specifically in the "MyAdventures" folder. This is because so your code can import itself to the server.

Additionally, navigate to your "AdventuresInMinecraft-PC" folder, and run "start.exe".

Lastly, open your Minecraft Launcher (Must have Minecraft purchased), switch the version to "1.12"(for compatibility requirements), "Start" the Game, navigate to "Multiplayer", and click "Direct Connect", and finally type in "localhost" to connect to your server.

PlayerPos & Teleport:

(*Finding your location, coordinates separating variables, and teleport.)

code: "pos = mc.player.getPos()" helps to find your location.

"x, y, z = mc.player.getPos()" alternatively, a nice way to get coordinates into seperate variables.

"mc.player.setPos(x+25, y, z)" specifies a particular location to teleport to (also combine "x, y, z = mc.player.getPos()" beforehand).


Set Block:

(*Placing a single block at a given set of coordinates.)

code: "x, y, z = mc.player.getPos()" and

"mc.setBlock(x+1, y, z, 1) to place a single block at a given set of coordinates.

10x10x10 cube solid stone:

(*As well as setting a single block with "setBlock" you can fill in a volume of space in one go with "setBlocks".)

code: "stone = 1 (stone is 1 as a variable)",

"x, y, z = mc.player.getPos()",

"mc.setBlocks(x+1, y+1, z+1, x+11, y+11, z+11, stone)"
